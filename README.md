# Hostel-Grocery-Management-System
Java-based inventory tracker for hostel grocery management using JDBC and MySQL.

## Features
- Inventory tracking
- Low-stock alerts
- Reporting

## Tech Stack
- Java
- MySQL
- JDBC

## How to Run
1. Import `hostel_grocery_db.sql` into your MySQL.
2. Update your JDBC credentials.
3. Run `Main.java`.
